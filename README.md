# Linked IT Jekyll Site

Jekyll blog and Azure MSP website for deployment to GitHub Pages.

## Local Development

Install Ruby and Bundler, then run:

```powershell
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

## GitHub Pages

This project uses the `github-pages` gem and standard Jekyll folders, so it can be deployed from a GitHub repository using GitHub Pages.
